# The Dragonfruitvoid

## Controls

- WASD - Move
- Mouse movement - Direction for all skills (including dodge)
- Left click - Shoot (hold down for best results)
- Space - Jump (short invulnerability to waves)
- V - Dodge
- E - Blink
- R - Portal (unimplemented)
- 4 - Pull
